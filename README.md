# 🚀 InfoBot (Public Edition)

Интерактивный торговый помощник: запустил лаунчер → получил веб-интерфейс, ботов, AI и понятную документацию. Всё заточено под «быстрый старт без лишних PDF».

---

## ✨ Что внутри

| Блок | Возможности |
| --- | --- |
| 🎛 **Управление** | Веб-панель, история действий, уведомления, восстановление состояний |
| 🤖 **Боты** | RSI/трендовые сценарии, break-even, трейлинг, ручной запуск/стоп |
| 🧠 **AI** | Подтверждение сигналов, автообучение, адаптивный риск-менеджмент |
| 🗄️ **База данных** | SQLite БД для хранения всех данных AI (автоматическая проверка целостности, резервное копирование) |
| ⚙️ **Сервис** | GUI-лаунчер, автоматическая установка зависимостей, экспорт/импорт конфигов |

---

## ▶️ Как запустить (3 шага)

1. **Ставим Python 3.11+** на Windows/macOS/Linux.
2. **Запускаем лаунчер**:  
   🪟 `start_infobot_manager.cmd` / `start_infobot_manager.vbs` • 🐧/🍎 `chmod +x start_infobot_manager.sh && ./start_infobot_manager.sh`
3. **Следуем мастеру**: создание `.venv`, установка зависимостей (TensorFlow, ccxt и т.д.), подключение биржи, запуск `app.py`/`bots.py`/`ai.py`.

> Лаунчер показывает прогресс, блокирует кнопки на длительных шагах и помнит состояние — повторный старт занимает пару секунд.

**Минимум железа:** 4 CPU / 8 ГБ RAM / 5 ГБ диска. Для AI — 16 ГБ RAM и стабильный интернет. Детали в `docs/INSTALL.md`.

## ❓ Мини-FAQ
- **API-ключи сразу нужны?** Нет. Пока ключей нет, работает демо-режим; подключить биржу можно позднее.
- **Можно торговать руками?** Да. Управляйте ботами, ставьте защиту, открывайте сделки вручную.
- **Как обновляться?** Скачайте свежую сборку и распакуйте поверх — лаунчер сам обновит окружение и зависимости.
- **Где хранятся данные AI?** Все данные AI хранятся в SQLite БД (`data/ai/ai_data.db`). БД автоматически проверяется на целостность при каждом запуске и создает резервные копии при необходимости.
- **Где поддержка?** Контакты и лог событий доступны прямо в лаунчере (плюс Telegram).
