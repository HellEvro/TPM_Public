# InfoBot Manager

`launcher/infobot_manager.py` — кроссплатформенный интерфейс для обслуживания проекта. Он позволяет:

- Создавать и обновлять виртуальное окружение `.venv`
- Устанавливать зависимости из `requirements.txt`
- Проверять и подтягивать обновления из Git
- Получать Hardware ID и инструкции по лицензии (`scripts/activate_premium.py`)
- Запускать и останавливать основные сервисы: `app.py`, `bots.py`, `ai.py`
- Открывать ключевые файлы (`app/config.py`, `app/keys.py`) и журналы

## Запуск

После выполнения установочного скрипта (`installer/install_<os>.{ps1|sh}`) используйте соответствующий лончер:

- Windows: `start_infobot_manager.cmd`
- Linux/macOS: `./start_infobot_manager.sh`

Лончер активирует `.venv` (если создан), запускает GUI и предоставляет доступ ко всем операциям из одной точки.

## Требования

- Python 3.9+ с поддержкой `tkinter`
- Git (опционально, для проверки обновлений)

Если `tkinter` отсутствует (например, в минимальных сборках Linux), установите пакет `python3-tk` или используйте полноценную сборку Python.
