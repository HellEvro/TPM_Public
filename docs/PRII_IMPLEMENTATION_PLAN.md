## Итог реализации

- **API ПРИИ:** `GET/POST /api/bots/prii-coin-params`, `GET/POST/DELETE /api/bots/prii-coin-params/<symbol>`, `POST /api/bots/prii-trades-analysis` (запуск анализа по расписанию).
- **Модуль изучения сделок:** `bots_modules/prii_trades_learner.py` — загрузка из `bot_trades_history`, оценка, запись только в `full_ai_coin_params`; запуск после закрытия позиции в ПРИИ и по API.
- **Оптимизатор:** при `full_ai_control` сохраняет результаты только в `full_ai_coin_params` (ai_strategy_optimizer.py).
- **Проверки:** `tests/verify_prii_plan.py` — проверки 10.1–10.3.
