- `docs/QUICKSTART.md` — все действия на 15 минут (лаунчер + ручной сценарий).

---

## 2. AI и премиальные модули

- `docs/AI_README.md` — единственный актуальный источник про AI (модули, лаунчер, данные, лицензии, тесты).
- `docs/AI_UI_CONFIGURATION.md` — настройки из Web UI, если требуется ручная правка параметров.
- `docs/AI_ML_PARAMETER_QUALITY_PREDICTOR.md` — **НОВОЕ:** ML модель для предсказания качества параметров, генерация оптимальных параметров.

## 3. Архитектура и процессы

- `docs/SYSTEM_OVERVIEW.md` — единый обзор сервисов, потоков конфигурации и хранения данных.

## 4. Дополнительные материалы

- `docs/INSTALL.md` — глубокая установка (с прокси, оффлайн зеркалами и проверками системных библиотек).
- `docs/AI_UI_CONFIGURATION.md` — таблицы параметров UI + подсказки по Smart Risk.
- `docs/CHANGELOG.md` — история правок (синхронизирована с `main`).

> Ранее существовавшие файлы вида `AI_*_SUMMARY`, `*_TEST_REPORT.md`, `ai_guides/*` и т. п. удалены как дублирующие. Вся ключевая информация перенесена в перечисленные выше документы.

---

## 5. Мини чеклист перед работой

1. `git pull` и `python scripts/sync_to_public.py`.
2. Проверка `python scripts/verify_ai_ready.py`.
3. Просмотрите `logs/ai.log` → нет ли ошибок лицензии/данных.
4. Ознакомьтесь с разделом Roadmap в `docs/AI_README.md`, чтобы понимать текущий приоритет (Pattern Recognition, мульти-биржи и т. д.).
