# GUI для работы с базами данных InfoBot

Полнофункциональное GUI приложение для работы с базами данных проекта InfoBot.

## Возможности

- ✅ **Автоматический поиск БД** - находит все `.db` файлы в проекте
- ✅ **Открытие внешних БД** - можно открыть любую SQLite базу данных
- ✅ **SQL редактор** - выполнение произвольных SQL запросов (F5)
- ✅ **GUI для CRUD** - добавление, редактирование, удаление записей через интерфейс
- ✅ **Просмотр таблиц** - удобный просмотр данных с пагинацией
- ✅ **Изменяемые размеры окон** - все панели можно изменять по размеру

## Запуск

### Windows
Двойной клик по `database_gui.vbs` или `database_gui.cmd`

### Linux/MacOS
```bash
./database_gui.sh
```

Или напрямую:
```bash
python3 scripts/database_gui.py
```

## Использование

### Автоматический поиск БД
При запуске автоматически находятся все базы данных в проекте:
- `data/bots_data.db` - основная БД ботов
- `data/ai/ai_data.db` - БД AI модуля
- `license_generator/licenses.db` - БД лицензий

Двойной клик по БД в списке открывает её.

### Открытие внешней БД
1. Нажмите "Открыть внешнюю БД"
2. Выберите файл `.db`
3. База данных откроется и появится в списке

### SQL редактор
1. Откройте базу данных
2. Введите SQL запрос в редактор
3. Нажмите F5 или кнопку "Выполнить запрос"
4. Результаты появятся в панели "Результаты SQL"

### Работа с таблицами

#### Просмотр данных
1. Откройте базу данных
2. Выберите таблицу из списка
3. Данные автоматически загрузятся

#### Добавление записи
1. Выберите таблицу
2. Нажмите "Добавить запись"
3. Заполните поля в диалоге
4. Нажмите "Сохранить"

#### Редактирование записи
1. Выберите запись в таблице
2. Нажмите "Редактировать"
3. Измените поля в диалоге
4. Нажмите "Сохранить"

#### Удаление записи
1. Выберите запись в таблице
2. Нажмите "Удалить"
3. Подтвердите удаление


## Горячие клавиши

- **F5** - Выполнить SQL запрос

## Требования

- Python 3.9+
- tkinter (обычно входит в стандартную установку Python)
- SQLite3 (встроен в Python)

## Особенности

- Все окна и панели изменяемы по размеру (paned windows)
- Автоматическое определение схемы таблиц
- Безопасное выполнение SQL запросов с параметрами
- Отображение статистики по базе данных
- Поддержка больших таблиц с пагинацией (показываются первые 1000 записей)
